## Overview

ORIA MVP is a full-stack monorepo application for a digital music and NFT marketplace.

### Key Features

- User authentication via Supabase
- Nexus blockchain integration
- Progressive Web App (PWA)
- Dark theme UI
- Mobile-first design

### Architecture

- **Backend**: Express.js + Supabase + Nexus
- **Frontend**: React + Vite + TailwindCSS
- **Database**: Supabase (PostgreSQL)

For detailed setup instructions, see the main [README.md](../README.md) in the root directory.
